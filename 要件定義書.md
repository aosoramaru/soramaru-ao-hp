## 1. プロジェクト概要

### 1.1 目的
VTuber「宙丸アオ」の活動情報・世界観・目標・ファンコミュニティ・グッズ販売導線を統合する公式ハブサイトを構築する。
ファン獲得・コミュニティ形成・スーパーファン育成・ブランド価値の向上を目的とする。

### 1.2 ターゲットユーザー
- **一次**: 既存ファン（流青群）— 配信・グッズ・イベント情報をワンストップで確認
- **二次**: 新規視聴者 — VTuberの世界観・人柄を知り、ファン化を促進
- **三次**: 企業・コラボ希望者 — 問い合わせ窓口として機能

### 1.3 サイトコンセプト
**「宇宙を駆ける配達員」** をテーマにした高級感あるダークテーマ。
ゴールド × ディープブルー × シアンを基調とした宇宙の世界観を表現。

## 3. 機能要件

### 3.1 共通機能
- **レスポンシブデザイン**: PC・タブレット・スマートフォン対応（ブレイクポイント768px）
- **モバイルナビ**: 全画面フェード表示、世界観演出あり、背景スクロール完全無効化
- **ニュースカルーセル**: 自動再生 + 手動スワイプ + ドットナビ、左右両方向スライドアニメーション
- **ローディング画面**: キャラクター画像アニメーション付き
- **背景エフェクト**: Canvas描画の星空 + 流れ星（全ページ共通）
- **スクロールリビール**: IntersectionObserverを用いたフェードインアニメーション
- **固定SNSリンク**: スクロール位置に関わらず画面左下に常時表示（モバイル）

### 3.2 ページ固有機能
- **Profile**: 衣装切り替え機能、ボイスサンプル再生（予定）
- **FAQ**: アコーディオン展開
- **Gallery**: ライトボックス拡大表示
- **Challenge**: 目標カード、進捗表示
- **Fan**: スーパーファン一覧グリッド

### 3.3 SNS連携
TikTok / X(Twitter) / Instagram / BOOTHへのリンク（全ページ共通）

## 4. 非機能要件

### 4.1 パフォーマンス
- 初回表示（LCP）: 2.5秒以内目標
- 画像最適化: WebP推奨（現状PNG/JPG）
- JavaScript/CSSの最小化

### 4.2 SEO要件
- 各ページに固有の`<title>`と`<meta description>`
- Open Graph / Twitter Cardメタタグ
- 構造化データ（JSON-LD）でPerson/Organization
- sitemap.xml / robots.txt
- Google Search Console登録

### 4.3 セキュリティ（実装済み）
- Content-Security-Policy
- X-Frame-Options: SAMEORIGIN
- Strict-Transport-Security (HSTS)
- X-Content-Type-Options: nosniff
- Referrer-Policy: strict-origin-when-cross-origin
- Permissions-Policy
- 外部リンクに`rel="noopener"`
- AI学習・生成への利用禁止明記

### 4.4 アクセシビリティ
- 適切なalt属性
- セマンティックHTML
- キーボード操作対応
- 色のコントラスト比確保

### 4.5 ブラウザ対応
- Chrome / Safari / Edge / Firefox 最新版
- iOS Safari / Android Chrome 最新版

1. ローカルで編集
2. `git commit` → `git push origin master`
3. Vercelが自動デプロイ（約10秒）

## 8. Google Search Console登録手順

1. [Google Search Console](https://search.google.com/search-console) にアクセス
2. 「プロパティを追加」→ URLプレフィックスで `https://soramaru-ao-hp.vercel.app` を入力
3. HTMLメタタグで確認を選択 → 発行された `<meta name="google-site-verification" content="XXXX">` を `index.html` の`<head>`に追加
4. 全ファイルをプッシュ → デプロイ後にSearch Consoleで「確認」ボタンを押す
5. サイトマップ送信: `sitemap.xml` を送信
